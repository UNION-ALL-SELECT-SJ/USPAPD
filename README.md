# USPAPD - Universal Serial Parallel & Analog Protocol Diagnostics Device
A human-portable self-contained Debugger and diagnostics tool for various common hardware communication protocols encountered in consumer, hobbiest and industrial electronics.

## Abstract
A portable device for tapping into various short-distance communication lines common in most smart electronic devices, such as computers, microcontrollers, sensors, robots etc.
The USPAPD is intended to be a handy tool for capturing information sent along these lines, and is aimed at hardware developers and engineers to peform quick, light-weight testing and debugging.

## Basic Workflow
- Intakes data from the following protocol types:
  - serial: SPI, I2C, UART, FDCAN (compatible with BxCAN and CAN 2.0), USB (as device)
  - Parallel: 8, 16 or 32 bit bus
  - Analog
- Displays real-time and historical data on built in display, configurable to several formats
  - decimal and hexidecimal integers
  - single and double precision floating point
  - UTF-8 (this includes ASCII)
- Can remember a large quantity of individual bytes of data for ready browsing
- Graphing and visualisation supported for numerical data
